# 🔢 Percentage Calculator

**The fastest free percentage calculator online** — no ads, no signup, instant results.

🔗 **Live site:** [ronaldolum.github.io/percentage-calculator](https://ronaldolum.github.io/percentage-calculator)

---

## What can it do?

- ✅ **What is X% of Y?** — e.g. What is 15% of 200? → 30
- ✅ **X is what % of Y?** — e.g. 30 is what % of 200? → 15%
- ✅ **Percentage change** — e.g. from 80 to 100 → +25%
- ✅ **Increase / Decrease by %** — e.g. 200 increased by 15% → 230
- ✅ Results update **instantly as you type**
- ✅ **Copy to clipboard** with one click
- ✅ Works on mobile, tablet, and desktop
- ✅ No dependencies, no frameworks — pure HTML/CSS/JS

## How to use locally

Just download `index.html` and open it in any browser. That's it — no install, no server needed.

```bash
git clone https://github.com/yourusername/percentage-calculator.git
cd percentage-calculator
open index.html
```

---

## Deploy your own

1. Fork this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Your site is live at `yourusername.github.io/percentage-calculator`
